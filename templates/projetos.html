Gerenciador de Senhas

Descrição: Este projeto tem como objetivo armazenar e criar as senhas do usuário em poucos cliques. Com uma senha principal para poder acessar o aplicativo, os dados contidos permanecem seguros. O usuário pode optar por uma senha de seu gosto ou o aplicativo se encarrega de randomizar uma senha com 10 caracteres sendo letras maiúsculas e minúsculas, números e caracteres especiais.

Framework: Flutter

Linguagem: Dart

Info: As senhas armazenadas ficam salvas apenas no dispositivo em um banco de dados SqLite (Sqflite) compatível com o Flutter.

Links: Repositório Github | Baixar aplicativo .apk
